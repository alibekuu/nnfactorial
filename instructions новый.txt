Инструкция по использованию программы управления системой кинотеатра
Программа позволяет управлять системой кинотеатра, предоставляя возможности добавления фильмов, регистрации пользователей, покупки и отмены билетов, а также удаления пользователей.

1. Запуск программы

Для запуска программы запустите программу test.py на Python, содержащий класс CinemaTicketSystem. После запуска программы будет отображено меню с возможными действиями.
В моем случае я использовал VisualCode c расширением Python. Затем запустил через терминал. Надеюсь все получиться.

2. Главное меню

После запуска программы вы увидите меню с следующими опциями:

1. Добавить новый фильм
2. Показать все доступные фильмы
3. Добавить нового пользователя
4. Показать всех пользователей
5. Удалить пользователя
6. Купить билет
7. Отменить билет
8. Выйти
Выберите опцию, введя соответствующую цифру и нажав Enter.

3. Описание функций программы

3.1. Добавление нового фильма
Выберите опцию 1 в меню.
Введите название фильма и нажмите Enter.
Программа сообщит вам, что фильм успешно добавлен.
3.2. Показ всех доступных фильмов
Выберите опцию 2 в меню.
Программа покажет список всех фильмов, доступных в системе.
3.3. Добавление нового пользователя
Выберите опцию 3 в меню.
Введите имя пользователя и нажмите Enter.
Программа сообщит вам, что пользователь успешно добавлен.
3.4. Показ всех пользователей
Выберите опцию 4 в меню.
Программа покажет список всех зарегистрированных пользователей.
3.5. Удаление пользователя
Выберите опцию 5 в меню.
Программа сначала покажет список всех пользователей.
Введите ID пользователя, которого хотите удалить, и нажмите Enter.
Программа сообщит вам, что пользователь успешно удален, или что пользователь с данным ID не найден.
3.6. Покупка билета
Выберите опцию 6 в меню.
Введите ID пользователя, который покупает билет, и нажмите Enter.
Введите ID фильма, на который покупается билет, и нажмите Enter.
Программа сообщит вам, что билет успешно куплен, или что введены неверные ID пользователя или фильма.
3.7. Отмена билета
Выберите опцию 7 в меню.
Введите ID билета, который хотите отменить, и нажмите Enter.
Программа сообщит вам, что билет успешно отменен, или что билет с данным ID не найден.
3.8. Выход из программы
Выберите опцию 8 в меню.
Программа завершит свою работу, и вы выйдете из системы.
4. Примечания

Все введенные ID должны быть числовыми значениями.
Программа отслеживает уникальные ID для фильмов, пользователей и билетов, которые автоматически генерируются при их добавлении.
При удалении пользователя все билеты, купленные этим пользователем, автоматически удаляются.
Пример использования программы
Запустите программу.
Введите 1 и добавьте фильм, например, "Inception".
Введите 3 и добавьте пользователя с именем "Alibek".
Введите 6, затем введите ID пользователя и ID фильма для покупки билета.
Введите 7 и отмените билет по его ID.
Введите 5, чтобы удалить пользователя, выбрав его ID из списка.
Таким образом, с помощью программы вы сможете эффективно управлять кинотеатром, регистрировать фильмы и пользователей, а также контролировать процесс покупки и отмены билетов.
